# 成績管理システム（Record Management System）

## 概要
本プロジェクトは、学生の成績情報を管理するための簡易的な成績管理システムです。  
学生・科目・成績といった情報をオブジェクトとして管理し、登録・参照・表示などの基本的な業務処理を想定して実装しています。

大学の講義・演習で学んだ **オブジェクト指向設計** を基に、  
実際の業務系システムを意識したクラス構成および責務分離を行いました。

---

## 主な機能
- 学生情報の管理
- 科目情報の管理
- 科目ごとの成績情報の管理
- 成績データの一覧表示
- CUI（コマンドライン）による操作
- GUI（画面）による操作
- データ構造を意識した処理設計

---

## 使用技術
- Java
- オブジェクト指向プログラミング
- インターフェースを用いた設計
- Adapterパターン
- CUI / GUI 両対応構成
